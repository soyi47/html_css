# 선택자 조합



+ 여러 선택자를 조합하여 스타일 규칙을 선언하는 방법.
+ 각 **선택자 조건을 모두 충족할 때, 스타일 규칙이 적용**된다.



1. 요소 + class

   ```css
   p.bar { ... }
   ```

   + 태그 `<p>` 이면서, class 속성으로 'bar'를 가지는 요소

2. 다중 class

   ```css
   .foo.bar { ... }
   ```

   + class 속성으로 'foo'와 'bar'를 모두 가지는 요소

3. id + class

   ```css
   #foo.bar { ... }
   ```

   + id 속성으로 'foo'를 가지고, class 속성으로 'bar'를 가지는 요소
